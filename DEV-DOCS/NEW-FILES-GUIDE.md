# New Files Guide

**Last Updated:** March 5, 2026  
**Status:** Draft

## Purpose

Explain where new docs and code files should live.

## Current State

DEV-DOCS is the canonical documentation hub.

## Adding Documentation

- Feature docs → `DEV-DOCS/features/`
- Implementation patterns → `DEV-DOCS/implementation/`
- Requirements → `DEV-DOCS/requirements/`

## Adding Code

- UI routes → `src/app/(admin)` or `src/app/(public)`
- API endpoints → `src/app/api/v1`
- Domain logic → `src/modules`
- Shared utilities → `src/lib`
